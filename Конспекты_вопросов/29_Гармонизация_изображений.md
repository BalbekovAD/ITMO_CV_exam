# 29. Методы гармонизации изображений

## Краткий ответ

Гармонизация изображений - это приведение вставленного объекта или области к визуальному стилю фона: освещению, цветовой температуре, контрасту, теням, шуму, резкости и перспективе. Задача возникает при compositing, inpainting, augmented reality и редактировании изображений. Методы варьируются от классической цветовой коррекции и Poisson blending до нейросетевых моделей с encoder-decoder архитектурой, attention и diffusion-based editing.

## Основные понятия

- Composite image: изображение, собранное из нескольких источников.
- Foreground/background: вставленный объект и фон.
- Color transfer: перенос цветовой статистики.
- Illumination consistency: согласование освещения.
- Shadow consistency: согласование теней.
- Boundary artifacts: видимые швы по границе вставки.
- Poisson blending: классический метод плавного сшивания через градиенты.
- Image-to-image translation: преобразование изображения в другой визуальный домен.
- Harmonization mask: маска редактируемого объекта.

## Алгоритмы, архитектуры и формулы

Простая цветовая гармонизация может использовать перенос среднего и дисперсии по каналам:

$$
x'_c = \frac{x_c - \mu_{\text{fg},c}}{\sigma_{\text{fg},c}}\sigma_{\text{bg},c} + \mu_{\text{bg},c}
$$

где `fg` - foreground, `bg` - фон или область вокруг объекта.

Poisson blending решает задачу согласования градиентов:

$$
\min_f \int_\Omega \lvert \nabla f - \nabla g \rvert^2,\quad
\text{при } f \text{ на границе } = \text{фон}
$$

Идея: сохранить градиенты вставляемого объекта внутри маски, но согласовать значения на границе с фоном. Это хорошо убирает швы, но не всегда исправляет глобальное освещение.

Нейросетевая гармонизация обычно строится как:

```text
composite image + mask -> encoder-decoder -> harmonized image
```

Маска сообщает модели, какую область можно менять. Фон должен сохраняться.

Типичные loss-функции:

- reconstruction loss для target harmonized image;
- perceptual loss;
- adversarial loss;
- color/style loss;
- background preservation loss;
- identity/semantic consistency loss для сохранения объекта.

Современные подходы могут использовать:

- attention между foreground и background;
- domain/style embeddings;
- transformer-блоки для глобального контекста;
- diffusion editing для реалистичной перерисовки объекта с учетом окружения;
- depth/normal estimation для согласования геометрии и света.

## Сравнения, плюсы и минусы

Классическая цветокоррекция:

- Плюсы: быстрая, интерпретируемая, не требует обучения.
- Минусы: не исправляет сложные тени, материалы и локальное освещение.

Poisson blending:

- Плюсы: хорошо скрывает границы и швы.
- Минусы: может менять цвета нежелательно, не решает семантическую несовместимость.

CNN/Transformer harmonization:

- Плюсы: учитывает контекст, может править сложные различия.
- Минусы: требует данных, может менять идентичность объекта или фон.

Diffusion editing:

- Плюсы: высокий реализм, может дорисовывать освещение и детали.
- Минусы: меньше гарантий точного сохранения объекта, возможна стохастичность результата.

## Типичные ошибки

- Сводить гармонизацию только к подбору яркости.
- Изменять весь кадр вместо только вставленного объекта.
- Не учитывать тени и направление света.
- Забывать про границу маски: даже хороший цвет объекта может выдавать монтаж по краям.
- Путать harmonization со style transfer: первая стремится к незаметной вставке, вторая может намеренно менять стиль.
